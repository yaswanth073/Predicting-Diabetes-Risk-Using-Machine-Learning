## Overview

This project builds and compares six supervised machine learning models to predict whether an individual has diabetes using self-reported health and lifestyle survey data from the CDC BRFSS 2015 survey. The goal was to see how accurately diabetes risk can be predicted without any lab tests — just answers from a standard health questionnaire.

All analysis is done in **R**.

## Preprocessing Steps

1. Removed **1,635 exact duplicate rows** → 69,057 rows
2. Downsampled both classes to **33,960 each** (50/50 balance) → 67,920 rows total
3. Converted 14 binary features to **R factors**
4. **Z-score standardized** all 7 continuous features
5. **Stratified 70/30 train/test split** → 47,544 train | 20,376 test

## Models

| Model | Training Data | Key Settings |
|---|---|---|
| Logistic Regression | Full (47,544) | Binomial GLM, all 21 features |
| Lasso (glmnet) | Full (47,544) | alpha=1, λ=0.00082 via 5-fold CV |
| Random Forest | Full (47,544) | 300 trees, mtry=4 |
| GBM | Full (47,544) | 500 trees, depth=3, shrinkage=0.05 |
| SVM | 10% sub-sample (4,756) | Linear kernel, cost=1 |
| Neural Network | 10% sub-sample (4,756) | 8 hidden units, decay=0.01 |

> **Note:** SVM and Neural Network use a 10% sub-sample for computational tractability.

## Results

| Model | Accuracy | Sensitivity | Specificity | F1 | AUC |
|---|---|---|---|---|---|
| **GBM** ⭐ | **74.9%** | **78.9%** | 70.8% | **0.758** | **0.827** |
| Logistic | 74.6% | 76.7% | 72.4% | 0.751 | 0.820 |
| Lasso | 74.5% | 76.8% | 72.3% | 0.751 | 0.820 |
| SVM | 74.4% | 76.3% | 72.6% | 0.749 | 0.818 |
| RandomForest | 74.1% | 78.5% | 69.7% | 0.752 | 0.814 |
| NeuralNet | 72.7% | 74.9% | 70.5% | 0.733 | 0.798 |

**GBM** is the recommended model — best AUC (0.827), best sensitivity (78.9%), and fewest missed diabetic cases (2,148 false negatives).

---

## Key Findings

- Diabetes risk can be predicted at **~75% accuracy** and **AUC ~0.83** using only survey data — no blood tests needed
- **GBM** is the best overall model across all metrics
- **Logistic Regression** was surprisingly competitive (AUC = 0.820), suggesting the feature-outcome relationship is largely linear
- **Top predictors:** General Health, BMI, Age, High Blood Pressure, Income — all align with known clinical risk factors
- **Sensitivity** (catching true diabetic cases) is prioritized over specificity for a real screening context

---

## EDA Highlights

- **Age** is one of the strongest predictors — prevalence rises from 8% in the youngest group to 61% in the oldest
- **HighBP** shows the biggest gap between classes — 67% of diabetic vs 37% of non-diabetic patients
- **Income & Education** both negatively correlate with diabetes risk, reflecting known socioeconomic health disparities
- **General Health** self-rating has a near-perfect linear relationship with diabetes prevalence
